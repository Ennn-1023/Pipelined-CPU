# MIPS Pipelined CPU
## 包含指令
1. Integer Arithmetic: add, sub, and, or, sll, slt, andi
2. Integer Memory Access: lw, sw 
3. Integer Branch: beq, j, jr
4. Integer Multiply/Divide: multu 
5. Other Instructions: mfhi, mflo, nop
